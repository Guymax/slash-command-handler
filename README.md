This is a simple handler for the new Slash Commands on Discord.

<h1> How To Use It</h1>

<ul>
    <li>1) Clone the repo, you can do it using git and typing `git clone https://github.com/Le-Roux-nard/slash-command-handler.git`</li>
    <li>2) Install the dependencies using `npm i` inside the right folder</li>
    <li>3) Get a bot token <i>![if you don't know how to do you can click here](https://github.com/reactiflux/discord-irc/wiki/Creating-a-discord-bot-&-getting-a-token)</i></li>
    <li>4) go to the Slash Commands documentation right [here](https://discord.com/developers/docs/interactions/slash-commands) to understand how Slash Commands needs to be declared</li>
    <li>5) Enjoy !</li>
</ul>
