This is a simple handler for the new Slash Commands on Discord.

# How To Use It
1. Clone the repo, you can do it using git and typing `git clone https://github.com/Le-Roux-nard/slash-command-handler.git`.
2. Install the dependencies using `npm i` inside the right folder.
3. Get a bot token *[if you don't know how to do you can click here](https://github.com/reactiflux/discord-irc/wiki/Creating-a-discord-bot-&-getting-a-token)*
4. Go to the Slash Commands documentation right [here](https://discord.com/developers/docs/interactions/slash-commands) to understand how Slash Commands needs to be declared
5. Start the bot using `node main.js`
